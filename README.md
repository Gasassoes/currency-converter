# Currency Converter - Конвертер валют

**Автор:** [Азалия Агзамова]

## Описание проекта

GUI-приложение для конвертации валют с использованием внешнего API. Позволяет конвертировать суммы между различными валютами, сохраняет историю конвертаций в JSON файл.

## Функционал

- ✅ Выбор валюты "из" и "в"
- ✅ Ввод суммы для конвертации
- ✅ Кнопка конвертации
- ✅ Таблица истории всех конвертаций
- ✅ Сохранение истории в JSON
- ✅ Загрузка истории из файла
- ✅ Проверка корректности ввода (положительное число)
- ✅ Актуальные курсы валют через API

## Как получить API ключ

1. Перейдите на **https://www.exchangerate-api.com**
2. Нажмите "Get Free Key" (бесплатный ключ)
3. Введите ваш email и подтвердите регистрацию
4. Скопируйте полученный API ключ
5. Вставьте его в код: `self.api_key = "ваш_ключ"`

## Примеры использования

### Пример 1: Успешная конвертация
1. Выберите валюту "USD" (Из) и "EUR" (В).
2. Введите сумму: `100` и нажмите "Конвертировать".
3. **Ожидаемый результат:** Программа покажет результат (например, "100.00 USD = 92.50 EUR") и добавит запись в таблицу истории.

### Пример 2: Ошибка валидации
1. Оставьте поле "Сумма" пустым и нажмите "Конвертировать".
2. **Ожидаемый результат:** Программа выдаст предупреждение: "Введите сумму для конвертации!"

### Пример 3: Отрицательная сумма
1. В поле "Сумма" введите `-50`.
2. **Ожидаемый результат:** Программа выдаст предупреждение: "Сумма должна быть положительным числом!"
